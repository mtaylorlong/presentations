What is OpenStreetMap?

  • Not just a map, data!
  • A lot more than streets!

Why OSM?

  • Work to build the best map/database
  • You get to use the data!
  • Help people around the world

What to Map

  • POIs
  • Your neighborhood or area you know
  • Humanitarian Projects (Haiti, Typhoon Haiyan, Ebola)
  • Task Sites
  • "Map what's on the ground"

OSM Data

  • Nodes, Ways, Relations
  • Tags
  • Change Sets
  • Comments

Where to Start

  • Create an account
  • Choose an editor
  • Select appropriate base data
  • Start Mapping!

Considerations...

  • "Don't tag for the renderer"
  • Don't break Relations
  • Bulk imports are...difficult
  • Beware the mailing lists
  • Survey vs. Armchair
