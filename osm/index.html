Why Use OSM?

  • Work to build the best map/database
  • You get to use the data!
  • Free, Open Source (ODbL)
  • Help people around the world (HOT)

What to Map

Map what's on the ground
  • Points of Interest (POI's)
  • Your neighborhood / area you know well
  • Humanitarian Projects (Tasking Manager)

OSM Data

  • Nodes, Ways, Relations
  • Tags (amenity=restaurant)
  • Change Sets
  • Comments

Where to Start

  • Create an account
  • Choose an editor (iD, Potlach 2, JOSM)
  • Select appropriate base data
  • Start Mapping!
